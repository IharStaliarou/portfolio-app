Меня зовут Игорь

Я изучаю web-разработку с использованием HTML, CSS и JavaScript на протяжение 6 месяцев. Есть опыт верстки лэндингов с базовым функционалом на flex/grid, формы отправки данных пользователя на электронную почту, реализация несложной анимации бургер-меню, слайдера.

На данном этапе обучения закончил курс по Web-разработке на Udemy от Ивана Петриченко и изучаю JS + React так же на его курсе. Закончил изучение основ JS и перешел к работе со страницами - 3 модуль(DOM, event).

После завершения курса по JavaScript + React последует изучение Node.js от Богдана Стащука.
